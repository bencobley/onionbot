### To start the API 
Script will run on boot. Otherwise SSH into Pi 
1. `ssh pi@onionpi`
2. `cd onionbot`
3. `python3 API.py`

### To use the web portal
1. Start a MAMP <mamp.info> local server environment
2. In your browser, navigate to `0.0.0.0:5000/portal`
3. Start collecting data!
